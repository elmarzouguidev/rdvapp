
## Requirements

-   PHP >= 8.1 (or higher)
-   Composer
-   MySQL

## Installation

1.  Clone the repository
2.  Run `composer install`

3.  Create a database and update the `.env` file
4.  Run `php artisan migrate --seed`
5.  run `php artisan key:generate`
6.  Run `php artisan serve`
